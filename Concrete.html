The Concrete Compression Conundrum

Tags: Multivariate regression • Variance inflation factor (VIF)

Concrete is the most widely used building material in the world. It is a mix of cement and water with gravel and sand. It can also include other materials like fly ash, blast furnace slag, and additives.

The compressive strength of concrete is a function of components and age. A sample of more than a 1000 different combinations was published ( Source ).

Data Preparation

The data was first loaded into a Pandas dataframe using the `pd.read_csv` function. The `info` and `shape` methods were used to check the number of rows and columns and to get information about the data types and missing values. The code then checked for missing values and removed any duplicate rows.

Data Exploration and Visualization

The code calculated basic statistics for each column using the `describe` method. Boxplot and distribution plots were created for each column to check for outliers and to visualize the distribution of the data. A heatmap was created to visualize the correlations among the input variables. The code also performed univariate analysis and plotted histograms for each column.

Data Modeling

The code split the data into training and testing sets using the `train_test_split` function. The training data was then scaled using the `StandardScaler` method. A regression analysis was performed on the training data using the OLS method and the variance inflation factor (VIF) was calculated to check for multicollinearity among the input variables. Finally, the model was used to predict the target variable on the training data.
